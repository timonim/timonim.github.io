Obituary Generator

Remembering the Departed
with Elegance and Grace

The Obituary Generator is a heartfelt tribute project designed to help users create personalized and beautiful obituaries for their loved ones. Developed using HTML, CSS, and JavaScript, this website aims to provide an intuitive, user-friendly experience during a sensitive and emotional time.

The Problem

Creating an obituary for a loved family member was a sensitive yet necessary task that left me unsatisfied with the existing solutions available online. Many websites offered clunky interfaces or lacked the design and cultural sensitivity needed to honor my loved one's memory appropriately.

Navigating Development
as a Designer

As a designer stepping into the world of development, a project like this is a challenge. To streamline the process, I adopted Git for version control, effectively managing my design iterations, merging code updates, and keeping a well-organized project structure. By embracing HTML, CSS, and JavaScript, I was able to create an appealing user interface and add interactivity while continuously improving the generator's functionality – all from the perspective of a designer making their foray into development.

The Solution

To overcome these challenges, I set out to design and develop an obituary generator that catered to both user experience and the nuances of the culture. By focusing on creating an intuitive interface, I aimed to provide a simple yet elegant solution for people looking to create meaningful tributes for their loved ones.

Next steps

Having built a strong foundation for my online obituary generator, I now aim to broaden its scope and create a more inclusive platform. By focusing on customizing the design and functionality of the tool to cater to various obituary types – religious, cultural, or personal preferences – I ensure that every user can create a meaningful, fitting tribute using my online obituary generator, no matter their specific needs.
